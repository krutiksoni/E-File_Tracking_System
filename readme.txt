E-File Tracking System Project for Summer Internship at DEV-IT

Database present in SQL folder

1). Download xampp, xampp control- Apache Server and myphpAdmin
2). Open xampp control and start Apache Server as well as MySql.
3). Go to localhost/myphpAdmin
3). Create a new myphpAdmin database and import the e_file_tracking_system.sql file in SQL folder to it.
4). Go to localhost/E-File_Tracking_System and start enjoying the software.
5). For passwords, go to the users database and find password corresponding to the username.
